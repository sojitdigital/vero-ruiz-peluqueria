# Proyecto: Sitio web Vero Ruiz Peluquería

Carpeta lista para abrir en VS Code y arrancar el desarrollo con Claude Code.

## Contenido

- `BRIEF.md` — toda la información del negocio, servicios, precios, equipo, testimonios y objetivo comercial
- `data/servicios.json` — servicios y precios en formato estructurado, listo para usar en el código
- `assets/README.md` — qué fotos faltan conseguir antes de cerrar el sitio

## Próximo paso

Abrir esta carpeta en VS Code y pedirle a Claude Code que arranque el desarrollo del sitio tomando como referencia `BRIEF.md`, usando la misma estructura que el sitio ya entregado a SHARP Peluquería (https://sharp-peluqueria.vercel.app/) pero con marca y contenido propios de Vero Ruiz.

Pendientes antes de publicar: confirmar el WhatsApp vigente y conseguir fotos reales (ver `assets/README.md`).
